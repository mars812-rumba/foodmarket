## 🔗 API Endpoints

Home.tsx использует следующие endpoints:

```bash
# Получить список ресторанов
GET /api/restaurants
Response: { restaurants: Restaurant[] }

# Получить меню ресторана
GET /api/restaurants/{restaurant_id}/menu?t={timestamp}
Response: { 
  restaurant_id: string,
  categories: string[],
  items: MenuItem[]
}
```

---

## 🎯 Как это работает?

### 1. Загрузка ресторанов
```typescript
useEffect(() => {
  fetchRestaurants();
}, []);

// Загружает список ресторанов и выбирает первый
```

### 2. Загрузка меню
```typescript
useEffect(() => {
  if (selectedRestaurant) {
    fetchMenuItems();
  }
}, [selectedRestaurant]);

// Загружает меню выбранного ресторана
// Строит категории и героев из меню
```

### 3. Отображение меню
```typescript
<HeroCarousel heroes={heroes} onChoose={(id) => setOpenCategory(id)} />
<BottomSheet
  category={categories.find((c) => c.id === openCategory) ?? null}
  items={items.filter((i) => i.category === openCategory)}
  photoTimestamp={photoTimestamp}
/>
```

- Меню загружается из API, а не из кода
