# 🍕 MENU MANAGER — Админка управления меню ресторана

## Что это?

**MenuManager** — это админка для управления меню блюд ресторана. Переделана из `RestManager.tsx` (была для мебели) в ресторанную тематику.

## Структура

```
src/
├─ pages/
│  ├─ Home.tsx (витрина ресторана)
│  ├─ admin/
│  │  ├─ MenuManager.tsx ← НОВАЯ АДМИНКА
│  │  ├─ RestManager.tsx (управление ресторанами)
│  │  └─ ...
│
├─ assets/
│  ├─ heroes/
│  │  ├─ hero_pizza.png (рендер пиццы)
│  │  └─ hero_burger.png (рендер бургера)
│  │
│  └─ menu/
│     ├─ potato_chicken_menu.png
│     ├─ sushi_menu.png
│     ├─ steak_menu.png
│     ├─ carbonara_menu.png
│     └─ drinks_menu.png
```

## Как использовать?

### 1. Открыть админку
```
http://localhost:5173/admin/menu
```

### 2. Выбрать ресторан
- Выпадающее меню в хедере
- Автоматически загружает меню выбранного ресторана

### 3. Добавить блюдо
Нажми кнопку **"+ Блюдо"** или карточку **"НОВОЕ БЛЮДО"**

### 4. Заполнить форму

#### Вкладка "ОСНОВНОЕ"
- **НАЗВАНИЕ** — название блюда (Маргарита, Чизбургер и т.д.)
- **КАТЕГОРИЯ** — 🍕 Пицца, 🍔 Бургеры, 🥩 Стейки, 🍝 Паста, 🍣 Суши, 🥤 Напитки, 🍗 Курица
- **ЦЕНА (฿)** — цена в батах
- **ВЕС / РАЗМЕР** — 350г, 500мл и т.д.
- **ОПИСАНИЕ** — состав, особенности

#### Вкладка "ИНГРЕДИЕНТЫ"
Опциональные ингредиенты (доп. сыр, бекон и т.д.)
- Нажми **"+ Добавить ингредиент"**
- Заполни название и цену
- Удали кнопкой **X**

#### Вкладка "ФОТО"
- Загрузи фото блюда (PNG, JPG)
- Система автоматически сохранит в папку ресторана

### 5. Сохранить
Нажми **"СОХРАНИТЬ"** — блюдо добавится в меню

## Типы данных

### MenuItem
```typescript
{
  id: string,              // уникальный ID (автогенерируется из названия)
  name: string,            // название блюда
  slug: string,            // URL-friendly версия названия
  category: string,        // категория (pizza, burger, steak и т.д.)
  price: number,           // цена в батах
  weight: string,          // вес/размер (350г, 500мл)
  image: string,           // путь к фото
  ingredients?: [          // опциональные ингредиенты
    {
      id: string,
      name: string,
      price: number
    }
  ],
  available: boolean,      // видимо ли блюдо в меню
  notes: string,           // описание/состав
  photos: {
    main?: string,         // основное фото
    gallery?: string[]     // галерея фото
  },
  updated_at?: string      // дата обновления
}
```

### Ingredient
```typescript
{
  id: string,              // уникальный ID
  name: string,            // название (Доп. сыр, Бекон)
  price: number            // цена добавки
}
```

## API Endpoints

### Получить меню ресторана
```
GET /api/restaurants/{restaurant_id}/menu
```

### Добавить блюдо
```
POST /api/restaurants/{restaurant_id}/menu
Body: MenuItem
```

### Обновить блюдо
```
PUT /api/restaurants/{restaurant_id}/menu/{menu_id}
Body: MenuItem
```

### Удалить блюдо
```
DELETE /api/restaurants/{restaurant_id}/menu/{menu_id}
```

### Загрузить фото
```
POST /api/restaurants/{restaurant_id}/upload-menu-photos
Body: FormData (photos, menu_id, restaurant_id)
```

## Функции

✅ **Добавление блюд** — создание новых позиций в меню
✅ **Редактирование** — изменение существующих блюд
✅ **Удаление** — удаление блюд из меню
✅ **Загрузка фото** — добавление изображений блюд
✅ **Ингредиенты** — управление опциональными добавками
✅ **Поиск** — поиск по названию блюда
✅ **Фильтр по категориям** — быстрый поиск по типу блюда
✅ **Переключение видимости** — скрыть/показать блюдо в меню
✅ **Выбор ресторана** — управление меню разных ресторанов

## Как это работает на фронте?

### Home.tsx (витрина)
```typescript
// Загружает меню ресторана
const [menu, setMenu] = useState([]);

useEffect(() => {
  fetch(`/api/restaurants/${restaurantId}/menu`)
    .then(r => r.json())
    .then(data => setMenu(data.items));
}, [restaurantId]);

// Показывает блюда в Bottom Sheet
// Пользователь выбирает ингредиенты и добавляет в корзину
```

### MenuManager.tsx (админка)
```typescript
// Загружает меню для редактирования
const [items, setItems] = useState<MenuItem[]>([]);

useEffect(() => {
  fetch(`/api/restaurants/${selectedRestaurant}/menu`)
    .then(r => r.json())
    .then(data => setItems(data.items));
}, [selectedRestaurant]);

// Админ может добавлять, редактировать, удалять блюда
```

## Советы

💡 **Название автоматически генерирует ID** — не нужно вводить вручную
💡 **Фото загружаются в папку ресторана** — система сама организует структуру
💡 **Ингредиенты опциональны** — можно добавить блюдо без них
💡 **Переключение видимости** — скрой блюдо, если его нет в наличии
💡 **Поиск работает в реальном времени** — начни печатать название
